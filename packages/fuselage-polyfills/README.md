## Install

<!--install-->

Add `@boolfly.chat/fuselage-polyfills` as a dependency:

```sh
npm i @boolfly.chat/fuselage-polyfills

# or, if you are using yarn:

yarn add @boolfly.chat/fuselage-polyfills
```

<!--/install-->
