## Install

<!--install(dev)-->

Firstly, install the peer dependencies (prerequisites):

```sh
npm i -D prettier

# or, if you are using yarn:

yarn add -D prettier
```

Add `@boolfly.chat/prettier-config` as a dependency:

```sh
npm i -D @boolfly.chat/prettier-config

# or, if you are using yarn:

yarn add -D @boolfly.chat/prettier-config
```

<!--/install(dev)-->
